# Mantra do Nome

O "Mantra do Nome" é uma ferramenta inovadora desenvolvida pela Código Astral, projetada para explorar o poder dos mantras associados aos nomes dos usuários. Utilizando princípios da numerologia e astrologia, este programa oferece um cálculo preciso do mantra específico baseado no nome do usuário, proporcionando insights e benefícios personalizados.

## Características

- **Cálculo do Mantra do Nome:** Utiliza algoritmos avançados de numerologia para calcular um mantra único para cada nome.
- **Explicação Detalhada:** Oferece uma explicação detalhada do significado do mantra e sua relação com o usuário.
- **GUI Simples e Intuitiva:** Com apenas 3 botões (Calcular Numerologia, Limpar e Saiba Mais) e dois campos (entrada e saída), a interface é fácil de usar.
- **Dicas de Uso:** Orientações sobre como e quando utilizar o mantra para maximizar seus benefícios.

## Como Usar

1. Inicie o programa e digite seu nome completo no campo de entrada.
2. Clique em "Calcular Numerologia" para processar seu nome e receber seu Mantra do Nome personalizado.
3. Explore as funcionalidades adicionais clicando em "Saiba Mais" para entender melhor as funcionalidades e objetivos do programa.

## Pré-requisitos

Para executar o "Mantra do Nome", é necessário ter o Python 3.12 instalado em seu sistema.
